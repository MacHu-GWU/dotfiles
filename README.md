## Bootstrap 新电脑

给新电脑配置开发环境的步骤：

### **安装 Starship** - 跨 shell 的提示符工具

```bash
curl -sS https://starship.rs/install.sh | sh
```

官网：https://starship.rs/

### **安装 mise-en-place** - 多语言版本管理和任务运行器

```bash
curl https://mise.run | sh
```

官网：https://mise.jdx.dev/
